# CDRobo

Degrees of Freedom		6DoF / 7DoF - Either one can be used
Payload				2.0kg (full-range) 4.0kg (mid-range)
Total weight			8.2kg
Max reach			902mm
Max translation speed		50cm/s

Sensors: 
 - Torque
 - Position
 - Voltage
 - Temperature
 - Accelerometer
 - Gyroscope
